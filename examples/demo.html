A Simple Rich-Text Editor

selection
  • ↑: move upwards
  • ↓: move downwards
  • ←: move leftwards
  • →: move rightwards
  • ⇧↑: extend selection upwards
  • ⇧↓: extend selection downwards
  • ⇧←: extend selection leftwards
  • ⇧→: extend selection rightwards
format
  • ⌘B: bold
  • ⌘I: italic
  • ⌘U: underline
  • ⌘K: font color
  • ⌘⇧–: decrease font size
  • ⌘⇧+: increase font size
  • ⌘[: decrease indent
  • ⌘]: increase indent
  • ⌘.: custom 1
  • ⌘/: custom 2
edit
  • ↩︎: break into paragraphs
  • ⇧↩︎: break into lines
